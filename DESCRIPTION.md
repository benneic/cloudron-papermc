PaperMC is a high-performance Minecraft server fork that aims to fix gameplay and mechanics inconsistencies. This Cloudron package provides:

- **PaperMC Server** — The most widely used, high-performance Minecraft server
- **Lightweight Web Console** — View server status, player count, and live console logs from your browser
- **Auto-Updates** — Automatically checks for new PaperMC builds every 6 hours and applies them
- **Kid-Friendly Defaults** — Pre-configured for creative mode, peaceful difficulty, PvP disabled, and whitelist enabled
- **Cloudron SSO** — Web panel secured with Cloudron's built-in single sign-on
- **Easy Configuration** — Edit server files directly via Cloudron's File Manager

Server data including worlds, plugins, and configuration files are stored in `/app/data/server/` and are fully backed up by Cloudron.
